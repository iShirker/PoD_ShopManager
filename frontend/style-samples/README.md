# Visual style samples

Ten HTML mockups showing different visual directions for the POD Manager app. **Design only** — not wired to the app.

## How to view

1. **From the repo:** open `index.html` in a browser (e.g. `frontend/style-samples/index.html`).
2. **Or** serve the folder locally, e.g. `npx serve frontend/style-samples` then visit `http://localhost:3000`.

Use the index page to open each sample and compare. Pick one (or mix ideas) to implement in the app.
